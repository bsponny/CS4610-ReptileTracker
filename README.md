### Set Up:
Using `yarn`

```bash
yarn
```
To install the dependancies needed, you'll run
```bash
yarn add -D prisma
```
```bash
cp .env.example .env
```
```bash
npx prisma db push
```
```bash
yarn db:generate
```
```bash
yarn add -D express
```
```bash
yarn add @types/express
```
To run the server,
```bash
yarn dev
```
